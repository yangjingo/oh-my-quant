# AKShare — A 股数据源 (Python SDK)

## 连接

无需 token，直接 pip 安装即可：

```bash
pip install akshare
```

```python
import akshare as ak
```

## 优势

- **免费无门槛**：无需注册、无 API Key、无积分限制
- **覆盖广泛**：股票、基金、债券、期货、宏观、新闻等
- **纯 Python**：直接 `pip install` 即可用

## 局限

- 依赖第三方网站（东方财富、新浪等），网络不稳定时会超时
- 无融资融券、龙虎榜、股东增减持等深度数据
- 无期权数据
- 部分接口可能需要 cookie（如集思录可转债）
