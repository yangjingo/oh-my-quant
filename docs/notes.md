# 投资原则与笔记

> 最后更新: 2026-05-29

---

## 核心原则

**P1 — 收益预期管理**
追求 5-10% 年化率即可，主要的资金放在公司 E1 更稳妥。不追求暴利，追求可持续。
*对应智慧：博格「市场收益 − 成本 = 投资回报」*

**P2 — 心态修炼**
锻炼长期主义心态，克制短期交易欲望。用时间换空间，而不是用频繁操作换收益。
*对应智慧：巴菲特「如果你不愿意持有十年，就不要持有十分钟」*

**P3 — 注意力分配**
珍惜注意力，不亲自选股选基。让 Agent / 系统去盯盘和筛选，人只做决策和复盘。
*对应智慧：芒格「能力圈 —— 我们有三个篮子：进、出、太难」*

**P4 — 组合韧性**
不单吊一个方向。权益 + 固收 + 现金三层都要有，确保在任何市场环境里都能活下来，不用被迫在低点卖出。
*对应智慧：达里奥「全天候 —— 最重要的是你的组合在不同经济环境下都能生存」；塔勒布「反脆弱 —— 风会熄灭蜡烛，却能让火越烧越旺」*

## 大师法则

从 8 位投资大师的智慧中提炼的行动规则，每条法则对应一个原则锚点。

| # | 大师 | 法则 | 一句话 | 关联原则 |
|---|------|------|--------|----------|
| 1 | **芒格** | 能力圈 | 不懂的东西放「太难」篮子，不要碰 | P3 |
| 2 | **巴菲特** | 安全边际 | 好东西太贵了也是坏投资 | P1 |
| 3 | **马克斯** | 周期意识 | 要知道钟摆现在在哪，而不是预测它要去哪 | P2 |
| 4 | **达里奥** | 全天候 | 组合要能在四种经济环境（增长/衰退 + 通胀/通缩）中都生存 | P4 |
| 5 | **博格** | 成本是唯一确定的 | 费率 1% × 30 年 = 一套房。底仓用低费指数基金 | P1 |
| 6 | **彼得·林奇** | 两分钟测试 | 不能在两分钟内说明白为什么持有 → 不持有 | P3 |
| 7 | **利弗莫尔** | 趋势跟踪 | 市场永远是对的。利润会照顾自己，亏损不会 | P2 |
| 8 | **塔勒布** | 反脆弱 | 问自己：跌 30% 是恐慌卖出还是现金加仓？让组合从波动中获益 | P4 |

### 法则运用场景

- **加仓前** → 检查法则 1（能力圈）、法则 2（安全边际）
- **市场狂热时** → 读一遍法则 3（周期意识）、法则 7（趋势跟踪）
- **调仓/再平衡时** → 检查法则 4（全天候）、法则 5（成本）
- **复盘持仓时** → 跑法则 6（两分钟测试）
- **极端行情时** → 回到法则 8（反脆弱）

## 基金分类体系

### 按资产类别

```
基金
├── 固收类（防御）
│   ├── 货币基金（余额宝 / 活期+）
│   └── 债券基金（纯债 / 混合债）
│
└── 权益类（进攻）
    ├── 指数基金
    │   ├── 宽基：覆盖全市场（沪深300、中证500、标普500）
    │   │   → 选总费率最低的，不做择时
    │   ├── 窄基：行业 / 主题 / 策略
    │   │   ├── 被动型 → 费率优先
    │   │   └── 增强型 → 在跟踪指数基础上做主动优化
    │
    ├── 主动基金
    │   → 核心是选人：历史战绩、最大回撤、风格稳定性
    │   → 警惕换手率 > 200% 的「伪主动」基金
    │
    └── QDII / 海外
        → 提供汇率分散 + 市场分散
```

### 按风险等级

| 等级 | 类别 | 预期年化 | 最大回撤预期 | 配置角色 |
|------|------|----------|-------------|----------|
| 低 | 货币基金 | 1-2% | ≈0% | 流动性 |
| 中低 | 债券基金 | 3-5% | -3% ~ -5% | 压舱石 |
| 中 | 宽基指数 | 6-10% | -20% ~ -30% | 底仓 |
| 中高 | 主动基金 / 窄基 | 8-15% | -30% ~ -40% | 卫星 |
| 高 | 个股 / 行业 ETF | ±? | -50%+ | 战术仓位 |

### 评价主动基金的 4 个维度

1. **人** — 基金经理从业年限（>5 年优先）、历史最大回撤、风格是否漂移
2. **业绩** — 3/5 年年化收益 vs 基准、牛熊市分别表现
3. **成本** — 管理费 + 托管费合计 < 1.5%，换手率 < 200%
4. **规模** — 太小（< 1 亿）有清盘风险，太大（> 100 亿）策略容量受限

## 资产配置框架

### 基准配置 (6:3:1)

```
60%  权益类（宽基指数 40% + 主动/窄基 20%）
30%  固收类（债券 20% + 货币 10%）
10%  现金 / 机会资金（等极端行情加仓用）
```

### 再平衡规则

- **时间维度**：每季度检查一次，偏离超过 5% 就调回
- **事件维度**：单只基金跌幅 > 20% → 审查是否触发基本面变化，无变化则视为加仓机会
- **纪律**：不在市场暴跌时恐慌卖出，不在市场狂热时追高加仓

## 自检清单

每季度做一次：

- [ ] 每只持仓能否通过「两分钟测试」？（能说清为什么持有）
- [ ] 权益 / 固收 / 现金比例是否偏离基准 > 5%？
- [ ] 有没有单一行业 / 主题暴露 > 50%？
- [ ] 主动基金换手率有没有异常的？（> 200% 亮黄牌）
- [ ] 如果明天大盘跌 30%，我是恐慌还是兴奋？
- [ ] 费率合计是否在可接受范围内？
