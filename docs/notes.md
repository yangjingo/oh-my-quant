## 核心原则 (2026-05-28)

**P1 — 收益预期管理**
追求 5-10% 年化率即可，主要的资金放在公司 E1 更稳妥。不追求暴利，追求可持续。

**P2 — 心态修炼**
锻炼长期主义心态，克制短期交易欲望。用时间换空间，而不是用频繁操作换收益。

**P3 — 注意力分配**
珍惜注意力，不亲自选股选基。让 Agent / 系统去盯盘和筛选，人只做决策和复盘。

## 基金分类框架 (2026-05-28)

### 1. 固收类
- 债券基金
- 货币基金

### 2. 权益类

**主动基金**
- 核心是选人：看历史战绩、最大回撤、投资风格稳定性
- 代表性持仓：华安动态灵活配置A (040015)

**指数基金**
- 宽基：覆盖整个市场（沪深300、中证500）
- 窄基：行业/主题/策略指数
  - 被动型：选管理费最低的
  - 增强型：在跟踪指数基础上做主动优化
